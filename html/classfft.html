Detailed Description

Class for performing fourier transforms usinf FFTW3.

Allows for the set-up of a fftw3 plan (the most time consuming part of the process) when the class instantiated, allowing the performance of forward and inverse fast fourier transforms with minimal computational cost inside the RtAudio Callback when the method are called.
